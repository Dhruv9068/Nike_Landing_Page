# Nike Landing Page 👟

A beautiful, modern, and responsive landing page showcasing Nike shoes with an elegant design featuring a vibrant green circular background element and smooth animations.

![Nike Landing Page](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![Responsive](https://img.shields.io/badge/Responsive-Yes-success?style=for-the-badge)

## 🌟 Features

- **Modern UI/UX Design**: Clean and minimalist interface with a focus on the product
- **Eye-catching Visual Elements**: Distinctive green circular background design element
- **Fully Responsive**: Optimized for all devices (desktop, tablet, and mobile)
- **Smooth Animations**: Hover effects and transitions for enhanced user experience
- **Social Media Integration**: Quick access links to Facebook, Instagram, and Twitter
- **Interactive Elements**: 
  - Animated "Buy Now" button
  - Hover effects on navigation links
  - Interactive social media icons
  - Shopping cart icon

## 🎨 Design Highlights

- **Color Scheme**: 
  - Primary: Vibrant Green (#7ed957)
  - Secondary: Dark Gray (#1b1b1b)
  - Background: Clean White (#fff)
  
- **Typography**: 
  - Main Font: 'Poppins' (modern and readable)
  - Accent Font: 'Lobster' (for stylized "Shoes" text)

- **Layout Features**:
  - Circular background design element positioned elegantly
  - Tilted shoe image for dynamic visual appeal
  - Strategic z-index layering for depth

## 🚀 Quick Start

### Prerequisites

No prerequisites required! This is a pure HTML/CSS project with no build tools or dependencies needed.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Dhruv9068/Nike_Landing_Page.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd Nike_Landing_Page
   ```

3. **Open in browser**
   - Simply open `Index.html` in your preferred web browser
   - Or use a local development server:
     ```bash
     # Using Python 3
     python -m http.server 8000
     
     # Using Python 2
     python -m SimpleHTTPServer 8000
     
     # Using Node.js (with http-server package)
     npx http-server
     ```

4. **View in browser**
   - Navigate to `http://localhost:8000` (if using a server)
   - Or directly open the HTML file

## 📱 Responsive Design

The landing page is fully responsive with breakpoints at:

- **Desktop (>1024px)**: Full layout with large typography and images
- **Tablet (768px - 1024px)**: Adjusted font sizes and image dimensions
- **Mobile (<768px)**: 
  - Stacked navigation
  - Centered content
  - Adjusted image positioning
  - Touch-optimized buttons and icons
